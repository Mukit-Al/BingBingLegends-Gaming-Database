🚀 Quick Start

Prerequisites: Microsoft SQL Server (2016+) and SQL Server Management Studio

Installation:

  1. Clone the repository
  2. Execute schema scripts in order (01_create_database.sql → 02_create_tables.sql → 03_insert_sample_data.sql)
  3. Run query files to explore the database capabilities
